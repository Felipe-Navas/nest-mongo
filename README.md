# Nest Mongo

A Nest.js app with mongoDB

## Description

A Nest.js app with mongoDB

## Getting Started

### Dependencies

- `npm`
- `node`

### Installing

- Clone the repository and install dependencies:

```
git clone https://github.com/Felipe-Navas/nest-mongo.git && cd nest-mongo && npm install
```

### Executing program

```
npm start
```

### Running tests

```
npm test
```
